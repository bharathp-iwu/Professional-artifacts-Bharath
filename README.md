## Overview

This is my personal academic portfolio built for my Data Analytics Engineering course submission at Northeastern University. It showcases two applied data science projects demonstrating end-to-end skills in machine learning, exploratory data analysis, and data visualization.

---

## Featured Artifacts

### 1. Satellite Lifespan Prediction Using Machine Learning
Supervised machine learning project predicting the operational lifespan of satellites based on design specifications and orbital parameters. Models include Random Forest, Gradient Boosting, and Linear Regression, evaluated using RMSE, MAE, and R².

**Tools:** Python, Scikit-learn, Pandas, Matplotlib, Seaborn, Jupyter Notebook

---

### 2. Instacart Market Basket Analysis & Customer Segmentation
Exploratory data analysis and unsupervised learning project on the Instacart Online Grocery Shopping dataset. Includes association rule mining (Apriori/FP-Growth) and K-Means customer segmentation with RFM analysis.

**Tools:** Python, Pandas, MLxtend, Scikit-learn, Tableau, Seaborn, Jupyter Notebook
